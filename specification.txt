CSS Properties Used:
1. color – Used to define text color.
2. background-color – Used to style backgrounds of sections and the navbar.
3. padding – Used to control internal spacing within elements.
4. margin – Used to manage spacing between elements.
5. border-radius – Used to create rounded elements such as buttons and containers.

JavaScript:
JavaScript is used to implement a scroll-to-top button that dynamically appears based on scroll position and allows smooth navigation back to the top of the page.

Bootstrap:
Bootstrap is used to create a responsive layout using its grid system and to style components such as the navbar, buttons, and spacing utilities.
